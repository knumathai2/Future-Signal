# Frontend — Outlook Signals

React + Vite + TypeScript + Tailwind CSS + Recharts.

## Setup

```bash
cd frontend
npm install
cp .env.example .env
```

## Development

```bash
npm run dev          # Vite dev server at http://localhost:5173
npm run lint
npm run format
npm run typecheck
```

## Build

```bash
npm run build         # production build, deployed to Vercel on git push
```

Screen implementation is tracked under `TASK-005` (`frontend/TASK-005-dashboard-skeleton`).
